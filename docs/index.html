Prayer Times

A native, free macOS menu bar app for Islamic prayer times.

Download latest release

Install via Homebrew

brew install --cask tareq1988/tap/prayer-times

The current build is ad-hoc signed (not yet notarized). If macOS blocks it, add --no-quarantine, or run xattr -dr com.apple.quarantine "/Applications/Prayer Times.app".

Features

  • Next prayer + live countdown in the menu bar (7 label styles)
  • Today's six times, iqamah, Liquid Glass panel (macOS Tahoe)
  • 8 calculation methods (Diyanet validated to ±1 min) + madhab + high-latitude rules
  • Per-prayer notifications, early reminders, and full Adhan playback
  • Location auto-detect; localized in English, العربية, Türkçe, বাংলা
  • Auto-updates via Sparkle

Requirements: macOS 14 Sonoma or later · Universal (Apple silicon + Intel)
